# Milestones

## M1 基础骨架

状态：已完成

目标：

- 初始化 `go.mod`
- 建立基础目录结构
- 提供最小 CLI 或 demo 入口
- 定义面向 Lua 5.1 子集的 `State`、`Value`、`Stack` 基础对象

交付标准：

- 项目可以编译
- 可以创建 VM 实例
- 可以跑一个最小演示入口

## M2 加载链路

状态：已完成

目标：

- 实现面向 Lua 5.1 子集的 lexer
- 实现面向 Lua 5.1 子集的 parser
- 生成 AST 或 IR
- 打通从源码到内部表示的转换

交付标准：

- 能读取并解析简单 Lua 脚本
- 能输出结构化的内部表示

## M3 执行核心

状态：已完成

目标：

- 实现 Lua 5.1 子集基础指令或节点执行
- 完成栈、帧、PC 管理
- 支持基础表达式与流程控制

交付标准：

- 能执行简单算术、变量、条件、循环脚本

## M4 运行时扩展

状态：进行中

目标：

- 支持 Lua 5.1 子集所需的函数调用
- 支持基础全局环境
- 提供最小宿主函数注册机制
- 逐步补充 table 或闭包能力

交付标准：

- Go 可以注入函数
- Lua 可以完成基本函数调用

## M5 验证与整理

状态：进行中

目标：

- 补齐 Lua 5.1 子集范围内的测试
- 增加样例脚本
- 整理错误处理
- 补齐开发文档

交付标准：

- 主链路具备稳定验证方式
- 项目对后续迭代者足够清晰
